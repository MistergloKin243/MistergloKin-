Comprendre la Compression Audio

Ce guide explique le fonctionnement du ratio et du seuil (threshold) dans un compresseur audio.

1. Le Seuil (Threshold)

Le seuil définit le niveau à partir duquel le compresseur commence à agir. Tout signal en dessous du seuil n’est pas touché.

Exemple :
- Seuil = -10 dB
- Signal entrant = -12 dB → aucune compression
- Signal entrant = -5 dB → dépassement de 5 dB → compression appliquée

2. Le Ratio

Le ratio indique combien le compresseur réduit le signal qui dépasse le seuil.

Formule :
Sortie au-dessus du seuil = Dépassement ÷ Ratio

Exemples avec différents ratios :

Dépassement (dB) Ratio 2:1 Ratio 5:1 Ratio 8:1
4 dB 2 dB passent 0,8 dB passent 0,5 dB passent
5 dB 2,5 dB passent 1 dB passe 0,625 dB passent
6 dB 3 dB passent 1,2 dB passent 0,75 dB passent
8 dB 4 dB passent 1,6 dB passent 1 dB passent

3. Interprétation des ratios

  • 1.5:1 → 2:1 : compression légère, naturelle, presque transparente
  • 3:1 → 4:1 : compression standard, équilibre entre contrôle et naturel
  • 5:1 → 8:1 : compression forte, son compact et percutant, idéal pour batterie et voix rap agressive
  • 10:1 et plus : limiteur, empêche le signal de dépasser le seuil

4. Exemple pratique

Ratio 2:1, seuil = -10 dB, signal dépasse de 5 dB → 2,5 dB passent
Ratio 5:1, seuil = -10 dB, signal dépasse de 5 dB → 1 dB passe
Ratio 8:1, seuil = -10 dB, signal dépasse de 6 dB → 0,75 dB passent
